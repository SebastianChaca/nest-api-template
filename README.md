# Nest js API template with Mongo DB

1. Open terminal at root folder
2. install project 
```
npm i
```
3. Create `.env` file using `.env.template` as guide
4. Data base 
 ```
   docker compose up -d
   ```
5. Start project
```
npm run start:dev
```
6. For documentation go to `/api#/`
